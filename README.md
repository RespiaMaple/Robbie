# Robbie
 
unity 2D平台遊戲試做

參考資料:https://www.youtube.com/watch?v=zTAc6QwU9vU&list=PL_Pb2I110MfGuXkePB0ZoMUa5FQ7mZU38&ab_channel=MStudio

操作方式: 移動(A、D或左右) 蹲下(S或下) 跳躍(Space，長按為大跳，蹲下再跳會更高) 靠近牆壁邊緣時會懸掛，esc暫停。


勝利條件:找到所有寶珠後會開門，進入門後即可獲勝
